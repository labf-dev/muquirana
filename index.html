Descrição do Aplicativo:

                    
MUQUIRANA é um programa desenvolvido para navegadores (browsers) que dispensa instalação, podendo ser acessado diretamente em dispositivos como tablets, celulares e computadores. Por suas características, é classificado como um mini-aplicativo, construído com as tecnologias HTML, CSS e JavaScript. Sua principal funcionalidade é auxiliar na organização de lançamentos de receitas e despesas de forma prática e acessível.

                    
                    
Funcionalidades:

                    
Título da Apuração - Campo destinado à inserção de um título que será utilizado nos extratos (exemplo: "Controle do mês de Janeiro de 2024").

                    
 Lançar Receita Permite registrar na memória do navegador os dados relacionados a uma receita, incluindo a data do crédito, uma descrição e o valor em R$.

                    
 Lançar Despesa Possibilita o registro na memória do navegador dos dados referentes a uma despesa, armazenando a data do débito, uma descrição e o valor em R$.

                    
 Extrato Exibe um relatório organizado com base no título da apuração e nos lançamentos de receitas e despesas realizados.

                    
 Exportar para PDF Permite exportar o extrato gerado para um arquivo no formato de impressão '.pdf'.

                    
 Salvar Dados Armazena os dados lançados (receitas, despesas e título da apuração) em um arquivo no formato '.json' para uso futuro.

                    
 Carregar Dados Recupera os dados previamente salvos no formato '.json', incluindo lançamentos de receitas, despesas e o título da apuração.

                    
 Resetar Dados Apaga todas as informações armazenadas no navegador, permitindo que novos lançamentos sejam realizados.

Dica Legal:

                                
Você pode abrir o MUQUIRANA várias vezes no mesmo navegador, e cada uma terá seus próprios dados separados e salvos. Para fazer isso:

                                
1. Crie uma cópia do arquivo muquirana.html.

                                
2. Dê um nome diferente para a cópia, como muquirana-01.html.

                                
3. Abra essa cópia em uma nova aba do navegador.

                                
Agora você terá duas versões do MUQUIRANA abertas, cada uma com seus dados separados.


                    
Importante:

                    
Se você mudar o nome do arquivo ou o local onde ele está salvo, o navegador vai criar um novo espaço de memória. Isso pode fazer parecer que os dados antigos desapareceram, mas eles ainda estão guardados no espaço de memória original.
